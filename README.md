# Viciouz Society Landing Page

This is a static deploy-ready version of the landing page.

## Files
- `index.html` — main site file
- `vercel.json` — optional Vercel config

## Upload to GitHub
1. Create a new GitHub repository.
2. Upload both files into the root of the repo.
3. In Vercel, import the GitHub repo.
4. Deploy.

No npm, Node, or terminal is required for this version.
